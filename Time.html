Select Maintenance > Time & Attendance > Time Recording to access this function

The Time Recording system provides comprehensive tools for tracking employee work hours, attendance, and schedule management. This module ensures accurate time tracking for payroll processing, compliance reporting, and operational efficiency within the organization.

Each time record contains essential information including clock-in/out times, break periods, overtime calculations, and attendance status. Proper time recording configuration is critical for maintaining accurate payroll and labor compliance.

The following table shows the time recording statuses and their corresponding meanings:

Status Code Description Time Type Calculation Method Payroll Impact
CLOCK_IN Employee Clock In Regular Time Start Time Tracking Begin Shift
CLOCK_OUT Employee Clock Out Regular Time End Time Tracking Complete Shift
BREAK_START Break Period Begin Unpaid Break Pause Time Tracking Deducted Hours
BREAK_END Break Period End Regular Time Resume Time Tracking Resume Paid Hours
OVERTIME Overtime Hours Overtime 1.5x Rate Calculation Premium Pay

Time Recording Configuration Process

  1. System Setup: Configure time recording parameters including shift schedules, break periods, and overtime rules.
  2. Employee Assignment: Assign employees to appropriate time recording groups and schedule templates.
  3. Device Configuration: Set up time clocks, biometric scanners, or mobile check-in systems.
  4. Rule Definition: Define attendance policies, late arrival rules, and overtime authorization procedures.
  5. Integration Setup: Connect time recording system with payroll and HR management systems.
  6. Reporting Configuration: Configure automated reports for attendance tracking and compliance monitoring.

Best Practices for Time Recording

  • Implement biometric or card-based systems to prevent buddy punching
  • Set up automated alerts for missed clock-ins or unusual time patterns
  • Regular calibration of time clocks to ensure accuracy
  • Provide employee training on proper time recording procedures
  • Maintain backup systems for time recording during system downtime
  • Regular audit of time records for compliance and accuracy
  • Secure access controls for time record modifications
